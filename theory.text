
a.	Que permet de faire la commande git pull ?
La commande git pull permet de faire un fetch d'un contenu et de le télécharger depuis un dépôt distant.
b.	Que permet de faire la commande git clone ?
La commande git clone est utilisé pour cibler un dépôt existant et créer une copie du dépôt cible.
c.	Expliquer ce que fait un « fork » ?
Un fork nous est utile pour qu'on puisse contribuer dans une repositoire
d.	Expliquez brièvement l’utilité d’une pull request (PR) ?
Une pull request, c'est pour la communication entre deux developpeurs. Par exemple, je demande à un autre développeur de faire un pull d'une branche de mon dépôt vers le don dépôt.
